# Security

## Reporting vulnerabilities in **this repository**

If you find a security issue in the scripts, configs, or documentation **in this repo** (e.g. accidental secret exposure, unsafe defaults in sample code), please open a private security advisory with GitHub or contact the repository owner.

Do **not** open public issues for undisclosed secrets.

## Scope

- **Freqtrade itself** — report security issues to the [upstream Freqtrade project](https://github.com/freqtrade/freqtrade).
- **Trading / exchange API keys** — never commit them; rotate keys if they were ever pushed.
