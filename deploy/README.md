# Deploy (optional)

This folder is a placeholder for server setup, env templates, or scripts when you promote a strategy beyond local Docker.

The **freqtrade-scalper** repo’s `deploy/` directory is a good reference for droplet-style flows; adapt names, ports, and secrets for this project.

Do **not** commit raw API keys or production passwords — use templates and `.env` (gitignored) or your host’s secret store.
